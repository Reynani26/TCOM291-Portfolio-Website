Here are some projects that I have worked on:

Mock Project Card Game

Mock Project for Saskatchewan Polytechnic that involved creating two card games (21 and Memory) using JavaScript, HTML and Bootstrap. Application was made with classmates for a project.

Reflection

Having this being the first experience developing software for a mock client, it was very overwhelming. There were a lot of poor decisions made from my group as a whole that made the first half the project very difficult. The biggest takeaway I took from the project was that communication as a team is very important to ensure that everyone was on the same page. Fortunately, we were able to pull it off and make a functioning application.

United Way Grant Management Application

Developed a web application for an external client that allows them to manage grant applications from non-profit organizations. Also allows for organizations to apply for grant applications.

Reflection

Being that this is my second time developing a software, I have much more experience with it and was a bit more prepared, but the kicker was that it was for an actual client. I definitely made mistakes during the process, but my communication with the team has improved. Working with clients that expect a fully functioning application does put a lot of pressure, but the biggest thing I tried to focus on was just enjoying the developing process. It was definitely cool to see a working application that I contributed to, that is going to be used in real time.

Workout Tracker Application

Developed a workout tracker application for my web development class using React frontend and Express API for backend. Project includes CRUD functionality and is a full stack application made by myself. I personally use this application.

Reflection

This was a project for my web development class. During the start of developing this project, it was very difficult as it was initially supposed to be a two-person project, which I did have a partner, but unfortunately decided to drop out of the course, in which I had to build it myself. I will say that I had a lot of fun working on this project, as working out is something I do and enjoy a lot, so building a workout tracker application made the developing process so much more fun. I use this application to track my workouts, and I hope to expand on it more depending on my needs
